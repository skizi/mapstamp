# mapstamp
Railsで作られたスタンプを地図上にデコることができる、位置情報共有アプリです。

## 技術構成
・サーバーサイドAPI : Rails  
・インフラ : nginx  
・Webフロントエンド : gulp, vue.js, babel, sass  
ruby 2.4.3  
Rails 5.1.6   

## できること
・イラストのデコレーション  
・テキストのデコレーション  
・装飾のデコレーション  
・フィルターのデコレーション  
・アニメーションさせることができる  
・TwitterとFaceBookへのシェア
