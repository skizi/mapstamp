# mapstamp
Railsで作られたスタンプを地図上にデコることができる、位置情報共有アプリです。

## 技術構成
・サーバーサイドAPI : Rails  
・インフラ : nginx  
・Webフロントエンド : gulp, babel, sass  

## できること
・イラストのデコレーション  
・テキストのデコレーション  
・装飾のデコレーション  
・フィルターのデコレーション  
・アニメーションさせることができる  
・TwitterとFaceBookへのシェア
